row 선택하기

  • Seires의 경우 []로 row 선택이 가능하나, DataFrame의 경우는 기본적으로 column을 선택하도록 설계
  • .loc, .iloc로 row 선택 가능
    • loc - 인덱스 자체를 사용
    • iloc - 0 based index로 사용
    • 이 두 함수는 ,를 사용하여 column 선택도 가능
