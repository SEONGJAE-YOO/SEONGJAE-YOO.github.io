  • sum - 데이터의 합
  • mean, std, var - 평균, 표준편차, 분산
  • min, max - 최소, 최대값

나이대별로 생존율 구하기

import math
def age_categorize(age):
    if math.isnan(age):
        return -1
    return math.floor(age / 10) * 10

df.set_index('Age').groupby(age_categorize).mean()['Survived']
