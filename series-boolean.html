Series 데이터 Boolean Selection으로 데이터 선택하기

학습목표

  1. Series boolean selection 활용하기
import numpy as np
import pandas as pd

Boolean selection

  • boolean Series가 []와 함께 사용되면 True 값에 해당하는 값만 새로 반환되는 Series객체에 포함됨
  • 다중조건의 경우, &(and), (or)를 사용하여 연결 가능
